Display current bitcoin (BTC/USD) price anywhere on your WordPress site online. 

== Description ==
Bitcoin Online shortcode displays current bitcoin price anywhere on your WordPress site. You can choose to display a current ticker prices like last, high, low, buy, sell, avg. Prices are fetched directly from mtgox.com using their data socket channel, so your site will not make ton of unnecessary requests to mtgox.com. **In the new version are available as well as data from bitstamp.net and btc-e.com too**.
== Installation ==
    * Download the plugin package and expand it.
    * Upload bitcoin_online to your plugins directory, likely wp-content/plugins/.
    * Activate the plugin through the Plugins menu in WordPress.
    * Visit the Bitcoin Online plugin options page to view samples of the usage
